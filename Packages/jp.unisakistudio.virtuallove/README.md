# 三点だいしゅきツール
Version 3.0.1

# 概要
三点トラッキングでも大好きなあの子とイチャイチャしたい！！
そんな人のためのVRSNS用アバターに使えるちょっとえっちなアニメーションファイルを２２種（男女計４４種）用意しました！！
I字バランスやJackOチャレンジもできます！

# このツールでできること
- 男女でペアになるアニメーションが２２種類。仲の良いあの子と一緒に使おう
- 鏡の前で使えば雰囲気が盛り上がること間違いなし！
- 足固定機能を使えば頭を動かすだけで腰が振れる！
- 首・足・腕すべてを固定すると、SpaceDragなどで自分のアバターを第三者視点で見て、撮影などを楽しめます

# プレゼント可能

# 使い方
- 同封の「VirtualLove_Boy.unitypackage」や「VirtualLove_Girl.unitypackage」をダブルクリックするかUnityでインポートしてください
- （一度インポートすれば次からはALCOMなどから「三点だいしゅきツール」をプロジェクトに追加することができます）
- 導入したいアバターを右クリックしてメニューから「ゆにさきスタジオ」の中にある「三点だいしゅきツール」の中から追加したいものを選択
- そのまま通常通りアバターをアップロード
- ゲーム内のExpressionsメニュー（メニューボタン長押し）で、姿勢を選択
- 不要な姿勢があれば「三点だいしゅき」のインスペクターから無効化や削除ができます
- 「可愛い座りツール」「三点だいしゅきツール」「添い寝ツール」を導入済みのアバターの場合そのままでは使用できないので、ツールの設定をアバターからを削除する機能があります。

# プレビルドについて
三点だいしゅきツールは、NDMFの機能を使用してアバタービルド時にメニューやアニメーションを生成していますが、
アバターをビルドするたびに処理を実行するとアバタービルドに時間がかかるようになってしまうため、
Ver3.0.0からプレビルド機能を追加いたしました。プレビルドを行うと三点だいしゅきツールのデータがModularAvatarのコンポーネントに変換されます。
アニメーションの設定などを変更した場合は再度プレビルドしないと、アバタービルド時に毎回処理が実行されてしまうので、こまめにプレビルドを行ってください。

# アニメーション調整機能について
同封のアニメーションは「あのん」ちゃんを使用して作成しているため、一部のアバターでは想定通りの姿勢にならないことがあります。
そのため、Ver3.0.0以降では姿勢のプレビューと調整を行う機能を導入いたしました。
三点だいしゅきオブジェクトのインスペクターより「アニメーション調整」ボタンを押すと、
シーンに配置してあるアバターをプレビューしながら姿勢を調整することができます。

# プリセットについて
三点だいしゅきツールVer3.0.0以降ではアバターに合わせて姿勢を調整するアニメーション調整機能が追加されています。
この機能を使って作成した調整アニメーションファイル、およびそれを設定した状態で作成したプリセットについては、

ご自身の自作アバター配付時などに是非共有してください。
